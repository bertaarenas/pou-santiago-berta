# Pou

Juego del Pou en el que tienes que cuidar a tu Pou gestionando su hambre, felicidad, energia e higiene o morira.
Ganas dinero jugando y lo puedes gastar en la tienda para comprar comida o jabón.

## Menú

- Comer - dale comida de tu inventario
- Dormir - suma energia
- Jugar - Juegos: piedra papel tijera o dados para ganar monedas
- Tienda - compra comida o jabón
- Estado - ver el estado0 actual
- Logros - ver logros desbloqueados
- Salir

El Pou muere si hambre, felicidad o higiene llegan a 0.
